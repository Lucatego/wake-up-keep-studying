# ¿Que necesitamos saber?
- Computer Vision
- Python
- Análisis de ojos - Eye Aspect Ratio (EAR) - Percentage of Eye Closure (PERCLOS)
- Sueño humano, tiempos de siestas ideales, cansancio y fatiga en expresiones faciales.

# ¿Cómo funcionará?
1. El programa usa la cámara para detectar tu expresión facial actual.
2. En caso detecte indices de fatiga o cansancio, el programa detendrá tu sesión de estudio (\*) para que puedas volver a concentrarte.
3. En caso te quedes dormido, el programa impondrá un mayor castigo o detendrá la sesión de estudio para asegurar un descanso ideal (\*\*).

# Anotaciones
(\*) El castigo puede ser un sonido de despertador, realizar actividad física o recomendaciones como tomar agua o lavarse la cara.

(\*\*) El descanso ideal tendrá que ser medido según estudios, por ejemplo, si el descanso ideal es de 30 minutos, el programa pondrá la alarma luego de 30 minutos de iniciado el sueño.
